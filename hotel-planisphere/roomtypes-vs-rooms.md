# 部屋タイプ（room_types）vs 部屋（rooms）の違い

## `room_types`テーブル（ベッドの種類）

| room_type_id | name     |
| ------------ | -------- |
| 1            | シングル |
| 2            | ツイン   |
| 3            | ダブル   |

## `rooms`テーブル（実際の部屋商品）

| room_id | name                 | room_type_id | capacity | area |
| ------- | -------------------- | ------------ | -------- | ---- |
| 1       | スタンダードシングル | 1            | 1        | 18   |
| 2       | デラックスシングル   | 1            | 1        | 25   |
| 3       | スタンダードツイン   | 2            | 2        | 28   |
| 4       | デラックスツイン     | 2            | 2        | 35   |
| 5       | プレミアムダブル     | 3            | 2        | 40   |

## 部屋と設備の多対多関係

### `facilities`テーブル

| facility_id | name         |
| ----------- | ------------ |
| 1           | ユニットバス |
| 2           | 独立洗面台   |
| 3           | 冷蔵庫       |
| 4           | 電子レンジ   |
| 5           | バルコニー   |

### `room_facilities`テーブル（中間テーブル）

| room_id | facility_id | 意味 |
| ------- | ----------- | ------------------------------------ |
| 1       | 1           | ← スタンダードシングルにユニットバス |
| 1       | 3           | ← スタンダードシングルに冷蔵庫       |
| 2       | 1           | ← デラックスシングルにユニットバス   |
| 2       | 2           | ← デラックスシングルに独立洗面台     |
| 2       | 3           | ← デラックスシングルに冷蔵庫         |
| 2       | 4           | ← デラックスシングルに電子レンジ     |
| 4       | 2           | ← デラックスツインに独立洗面台       |
| 4       | 3           | ← デラックスツインに冷蔵庫           |
| 4       | 5           | ← デラックスツインにバルコニー       |

## なぜ多対多なのか？

- **一つの部屋に複数の設備**: デラックスシングルには「ユニットバス」「独立洗面台」「冷蔵庫」「電子レンジ」
- **一つの設備が複数の部屋に**: 「冷蔵庫」はスタンダードシングル、デラックスシングル、デラックスツインにある

このように、部屋と設備は互いに多対多の関係になるため、中間テーブル`room_facilities`が必要になります。
