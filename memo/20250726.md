# 演習問題

以下の SQL のパフォーマンスを改善せよ

## 1

```sql
SELECT \* FROM employees WHERE birth_date BETWEEN '1960-01-01' AND '1969-12-31';
```

117138 rows in set (0.173 sec)

```sh
CREATE INDEX idx_birth_date ON employees(birth_date);
```

## 2

```sql
SELECT \* FROM titles WHERE title = 'Senior Engineer';
```

97750 rows in set (0.155 sec)

```sql
CREATE INDEX idx_title ON titles(title);
```

97750 rows in set (0.189 sec)

## 3

```sql
SELECT e.first_name, e.last_name, d.dept_no FROM employees AS e INNER JOIN dept_manager AS d ON e.emp_no = d.emp_no WHERE d.dept_no = 'd003';
```

17786 rows in set (0.056 sec)

```sql
CREATE INDEX idx_dept_manager_covering ON dept_manager(dept_no, emp_no);
```

17786 rows in set (0.057 sec)
